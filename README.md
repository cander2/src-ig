# Regulatory Correspondence FHIR Implementation Guide

This repository contains a FHIR Implementation Guide (IG) for standardizing structured regulatory correspondence from health authorities (e.g., FDA, EMA, PMDA) to market authorization holders (MAHs). Regulatory correspondece includes Health Authority Questions and the sponsor's Response to Health Authority Questions.

Health Authority Questions and Response to Questions are referred to by different names in different countries. For example, they are called Information Requests by US FDA or List of Questions by EMA.
